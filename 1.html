Your Assignment: Calibrating the Robot Sensors

Welcome to the team! Our company builds advanced automated robots to help businesses take over decision-making tasks at their front doors. But before our robots can use their brains to make decisions, they need to be able to "see" and store information correctly. Your job is to study our clients' rules, look closely at the people in the pictures, and write Java variables to save their characteristics using the correct data types.

Official Programming Rules

As you write code to digitize each person's features, remember our most important rule: Think of a variable as a labeled empty box. The contents of the box will change depending on which person the robot looks at, but the box label and data type stay exactly the same.

📦 Naming Rules

Create neat, descriptive variable names using lower camelCase architecture (e.g., hasTicket or guestAge).

💡 Choosing Types

Pick your data type carefully: use int for whole numbers, double for decimals, and boolean for true/false answers.

🔒 Handling Text

When saving strings of text, make sure to capitalize the String keyword and put your value inside double quotation marks "".

How your final code should look: 
// === TARGET 1 CHARACTERISTICS ===
[data_type] [yourVariableName] = [value_from_picture];
[data_type] [anotherVariable]  = [value_from_picture];
