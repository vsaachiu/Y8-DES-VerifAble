Project Phase 2: Interactive Communication

Until now, our robots have only been able to read static data. It's time to activate their microphone and communication arrays! By utilizing Java's Scanner tool, you will program our Sentinel units to ask human targets questions, listen to their answers, store those answers in variables, and speak back to them. Review the template guide, then complete the 7 client tickets below.

Mastering the Universal Template

Every interactive robot interaction follows a strict 5-step protocol. You will use this exact template in JDoodle to build every client solution. Study the structure below before moving to your first ticket. 

import java.util.Scanner; // Required to let our robot read user input

public class Main {
    public static void main(String[] args) {
        
        // 1. 🟨 VARIABLE DECLARATION (Create the empty boxes you need here)
        // Decide what data types you need and give them professional labels.
        // Example: int guestAge;
        
        // 2. INITIALIZE THE SCANNER (Turn on the robot's microphone)
        Scanner robotInput = new Scanner(System.in);
        
        // 3a. PROMPT THE USER (Ask the target a question)
        System.out.println("🤖 [VerifAble Sentinel Unit - ONLINE]");
        // Example: System.out.println("Please enter your age:");
        
        // 3b. ASSIGNMENT (Record their spoken answer into your box)
        // The tool must match the box! (e.g., nextLine(), nextInt(), nextDouble(), nextBoolean())
        // Example: guestAge = robotInput.nextInt();
        // REPEAT 3 and 4 for all needed information
        
        // 4. COMBINED OUTPUT (Speak back to the user)
        System.out.println("----------------------------------------");
        // Use the '+' symbol to stitch text and variables together.
        // Example: System.out.println("Access granted. Target is " + guestAge + " years old.");
        
        // Clean up our scanner stream
        robotInput.close();
    }
}

Client Ticket 01

The Lobby Greeter

🏬 Client: Corporate Headquarters

Mission Briefing

A major corporate headquarters wants a friendly robot stationed in their main lobby. When a guest walks up, the robot must ask for their name, record it into memory, and then output a customized, friendly greeting welcoming them to the building.

Target Data Requirements:

To complete this ticket in JDoodle, copy the Universal Template and follow these tutorial steps:

  • 1
    Variable Declaration: The robot needs a labeled box to store a person's name. A name is a collection of alphanumeric text characters. Choose the correct text data type and invent a good variable name (like guestName).
  • 3
    Prompt: Use System.out.println to print a message asking the user for their name.
  • 4
    Assignment: You need to read text. Use the command robotInput.nextLine(); and assign it to the variable box you made in step 1.
  • 5
    Output: Print a message that says "Welcome to the building, [Name]!" Make sure to use the + sign to connect your text string and your variable!
Quick Guide 1: Prompting the user with System.out.println Click to expand

Use a clear question so the user knows exactly what to type.

System.out.println("Please enter your name:");

Tip: Put prompts right before the input line that reads the answer.

Quick Guide 2: Reading input with Scanner (and converting types) Click to expand

You can read each data type directly, or read text with nextLine() and convert.

Scanner robotInput = new Scanner(System.in);

String name = robotInput.nextLine();
int age = robotInput.nextInt();
double temp = robotInput.nextDouble();
boolean isVip = robotInput.nextBoolean();
// If you read everything using nextLine():
String ageText = robotInput.nextLine();
int age = Integer.parseInt(ageText);

String tempText = robotInput.nextLine();
double temp = Double.parseDouble(tempText);

String vipText = robotInput.nextLine();
boolean isVip = Boolean.parseBoolean(vipText);
Quick Guide 3: Combining text + variables in output Click to expand

Use + to join static text and dynamic variable values.

String guestName = "Alex";
int ticketCount = 12;

System.out.println("Welcome, " + guestName + "!");
System.out.println("You currently have " + ticketCount + " tickets.");

Pattern: "text" + variable + "more text"

import java.util.Scanner; // Required to let our robot read user input

public class Main {

        // 1. 🟨 VARIABLE DECLARATION (Create the empty boxes you need here)
